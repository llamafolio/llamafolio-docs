In this tutorial we will create an adapter for Aave V2.

Create a new folder in `/src/adapters` and create an index file in it: `index.(js|ts)`.

Export your adapter defintion:

<Tabs>
<TabItem value="js" label="JavaScript">

```js title="/src/adapters/aave-v2-tuto/index.js"
const adapter = {};

module.exports = adapter;
```

</TabItem>
<TabItem value="ts" label="Typescript">

```ts title="/src/adapters/aave-v2-tuto/index.js"
import { Adapter } from "@lib/adapter";

const adapter: Adapter = {};

export default adapter;
```

</TabItem>
</Tabs>
